# APK 构建方法大全

除了之前介绍的方法外，还有以下几种构建 APK 的方式：

---

## 方法 1：使用 GitHub Actions（云端构建，推荐）

### 优点
- ✅ 完全免费
- ✅ 无需本地配置环境
- ✅ 自动构建，每次代码更新自动构建
- ✅ 生成可下载的 APK

### 步骤

1. **将项目上传到 GitHub**
   - 在 GitHub 创建新仓库
   - 上传代码到仓库

2. **创建构建配置文件**
   - 在项目中创建 `.github/workflows/build.yml`

3. **自动构建**
   - 每次 push 代码自动触发构建
   - 在 Actions 页面下载 APK

### 详细教程

[查看 GitHub Actions 构建教程](https://github.com/)

## 方法 3：使用 VS Code + 插件

### 优点
- ✅ 轻量级 IDE
- ✅ 界面简洁
- ✅ 插件丰富

### 步骤

1. **安装 VS Code**
   - 下载：https://code.visualstudio.com/

2. **安装必要插件**
   - "Flutter"（包含 Dart/Android 开发支持）

4. **打开项目并构建**
   - File → Open Folder → 选择 android-app
   - 按 F1 → 输入 "Flutter: Build APK"

## 方法 4：使用 Docker 容器

### 优点
- ✅ 环境隔离，不影响系统
- ✅ 可复现的构建环境
- ✅ 跨平台

### 步骤

1. **安装 Docker Desktop**
   - 下载：https://www.docker.com/products/docker-desktop

2. **创建 Dockerfile**
   ```dockerfile
   FROM ubuntu:20.04
   
   RUN apt-get update && apt-get install -y \
       openjdk-17-jdk \
       wget \
       unzip
   
   ENV ANDROID_SDK_ROOT=/opt/android-sdk
   ENV PATH=$PATH:$ANDROID_SDK_ROOT/cmdline-tools/latest/bin:$ANDROID_SDK_ROOT/platform-tools
   
   # 下载 Android SDK
   RUN wget -q https://dl.google.com/android/repository/commandlinetools-linux-9477386_latest.zip && \
       unzip -q commandlinetools-linux-9477386_latest.zip && \
       mkdir -p $ANDROID_SDK_ROOT/cmdline-tools && \
       mv cmdline-tools $ANDROID_SDK_ROOT/cmdline-tools/latest && \
       rm commandlinetools-linux-9477386_latest.zip
   
   # 安装 SDK 组件
   RUN yes | sdkmanager --licenses && \
       sdkmanager "platform-tools" "platforms;android-33" "build-tools;33.0.0"
   
   WORKDIR /app
   COPY . .
   
   CMD ["./gradlew", "assembleDebug"]
   ```

3. **构建镜像**
   ```bash
   docker build -t android-builder .
   ```

4. **运行构建**
   ```bash
   docker run -v $(pwd):/app android-builder
   ```

## 方法 6：使用 Makefile（自动化构建）

### 优点
- ✅ 一键构建
- ✅ 可定制构建流程
- ✅ 适合开发者

### 创建 Makefile

```makefile
.PHONY: build clean install

build:
	@echo "正在构建 APK..."
	./gradlew assembleDebug
	@echo "构建完成！"
	@echo "APK 位置: app/build/outputs/apk/debug/app-debug.apk"

clean:
	./gradlew clean

install:
	adb install app/build/outputs/apk/debug/app-debug.apk

run:
	adb shell am start -n com.phoneaudio.phoneaudioservice/.MainActivity
```

### 使用方法

```bash
# 安装 make（Windows 需要安装 chocolatey）
choco install make -y

# 构建
make build

# 清理
make clean

# 安装到手机
make install
```

## 方法 8：使用手机直接编译（Termux）

### 优点
- ✅ 纯手机操作
- ✅ 无需电脑

### 步骤

1. **安装 Termux**（F-Droid 下载）

2. **安装开发环境**
   ```bash
   pkg update
   pkg install openjdk-17
   pkg install android-sdk
   ```

3. **克隆项目**
   ```bash
   pkg install git
   git clone 项目地址
   ```

4. **构建 APK**
   ```bash
   cd android-app
   chmod +x gradlew
   ./gradlew assembleDebug
   ```
